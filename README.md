## Termux installation 
*`apt update && apt upgrade`

*`pkg install nodejs-lts`

*`git clone https://github.com/princekrvert/Temp-mail.git`

*`cd Temp-mail`


*`npm install `

*`chmod +x *`

*`node temp.js -u <username>`
